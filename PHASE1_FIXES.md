# Phase 1 Fixes — Applied

All changes committed to `index.html` and `sw.js`.

## index.html

### 1. `crypto.randomUUID()` — collision-safe channel IDs
**Before:**
```js
let _uid = Date.now()
function newId() { return 'ch_' + (++_uid) }
```
**After:**
```js
function newId() {
  if (typeof crypto !== 'undefined' && crypto.randomUUID) return 'ch_' + crypto.randomUUID()
  return 'ch_' + Date.now().toString(36) + Math.random().toString(36).slice(2)
}
```

### 2. `AbortController` 20s timeout — prevents fetch hang
**Before:**
```js
const r = await fetch(importUrl.value)
```
**After:**
```js
const _ctrl = new AbortController()
const _timer = setTimeout(() => _ctrl.abort(), 20000)
let r
try { r = await fetch(importUrl.value, { signal: _ctrl.signal }) }
finally { clearTimeout(_timer) }
```

### 3. `clearAll()` — replaced `confirm()` with Vue modal
**Before:**
```js
function clearAll() {
  if (!confirm(t('confirmClearAll'))) return
  channels.value = []; persist()
  showToast(t('allCleared'))
}
```
**After:**
```js
function clearAll() { modal.clearConfirm = true }
function confirmClearAll() {
  channels.value = []; persist()
  modal.clearConfirm = false
  showToast(t('allCleared'))
}
```

### 4. Duplicate i18n keys removed
Removed second occurrences of `language`, `theme`, `tvgUrl` in both SK and EN message objects.

### 5. `modal.clearConfirm` HTML block added
New `<div v-if="modal.clearConfirm">` alertdialog modal added alongside existing Delete Confirm modal.
